# Reference / Aesthetic Locks

The prompts in this repo reference image files that are **not committed here** — they are third-party game screenshots used only as a private aesthetic lock during generation, and redistributing them in a public repo isn't appropriate.

If you want to reproduce the look, the aesthetic targets are:

- **MapleStory** — for the chunky pixel GUI, panel-within-panel frames, and segmented stat bars.
- **Octopath Traveler** — for the warm HD-2D character-screen layout and atmospheric palette.

Drop your own reference screenshots in this folder under these names and the prompts will pick them up:

- `maplestory-reference.jpeg`
- `octopath-reference.jpeg`

The prompts also mention a `style-exemplar.png` — that was an earlier, off-target generation kept around as a "do **not** drift here" anti-example, and is intentionally excluded.
